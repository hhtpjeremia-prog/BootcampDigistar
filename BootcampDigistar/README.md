\# Digistar - House Price Prediction App



A simple Streamlit application for predicting house prices using a trained machine learning model.



\## Project Structure

\- app.py                : Streamlit application

\- house\_price\_model.pkl : Trained regression model

\- feature\_columns.pkl   : Feature list for inference

\- train.csv             : Training dataset



\## How to Run

1\. Create virtual environment

2\. Install dependencies:

&nbsp;  pip install -r requirements.txt

3\. Run the app:

&nbsp;  streamlit run app.py
